Nuevas Características de Visual Studio Code y Sublime Text 4

Algunas de las Nuevas Características de Sublime Text 4 (Build 4143)

¿Qué es Sublime Text 4?

Es un editor de texto y editor de código fuente. Está escrito en C++ y Python para los plugins. Desarrollado originalmente como una extensión de Vim, con el tiempo fue creando una identidad propia. Aún conserva un modo de edición tipo vi llamado Vintage mode.

Características de Sublime Text 4

  • Representacion GPU Sublime Text ahora puede utilizar su GPU en Linux, Mac y Windows al renderizar la interfaz. Esto da como resultado una interfaz de usuario fluida hasta resoluciones de 8K, todo mientras usa menos energía que antes.
  • Pestañas de Selección Mútiple Las pestañas de archivos se han mejorado para hacer que las vistas divididas sean sencillas, con soporte en toda la interfaz y comandos integrados. La barra lateral, la barra de pestañas, Ir a cualquier cosa, Ir a definición, Autocompletar y más se han modificado para que la navegación por el código sea más fácil e intuitiva que nunca.
  • Autocompletar Según El Contexto El motor de autocompletado se ha reescrito para proporcionar terminaciones inteligentes basadas en el código existente en un proyecto. Las sugerencias también se complementan con información sobre su tipo y proporcionan enlaces a definiciones.
  • Interfaz de Usuario Actualizada Los temas Predeterminado y Adaptativo se han actualizado con nuevos estilos de pestañas y atenuación del panel inactivo. Los temas y los esquemas de color admiten el cambio automático al modo oscuro. El tema adaptable en Windows y Linux ahora cuenta con barras de título personalizadas.
  • API de Python Actualizada La API de Sublime Text se ha actualizado a Python 3.8, al tiempo que mantiene la compatibilidad con versiones anteriores de los paquetes creados para Sublime Text 3. La API se ha ampliado significativamente, agregando funciones que permiten que complementos como LSP funcionen mejor que nunca. Lea la documentación renovada aquí.

Visual Studio Code Versión 1.75.1

¿Qué es Visual Studio Code?

Es un editor de código fuente desarrollado por Microsoft para Windows, Linux, macOS y Web. Incluye soporte para la depuración, control integrado de Git, resaltado de sintaxis, finalización inteligente de código, fragmentos y refactorización de código.

Características de Visual Studio Code

  • La depuración de declaraciones de impresión es cosa del pasado. Código de depuración directamente desde el editor. Inicie o conéctese a sus aplicaciones en ejecución y depure con puntos de interrupción, pilas de llamadas y una consola interactiva.
  • Comandos Git incorporados. Trabajar con Git y otros proveedores de SCM nunca ha sido tan fácil. Revisa las diferencias, prepara los archivos y realiza confirmaciones directamente desde el editor. Empuje y extraiga de cualquier servicio SCM alojado.
  • Extensible y personalizable. ¿Quieres aún más funciones? Instale extensiones para agregar nuevos idiomas, temas, depuradores y para conectarse a servicios adicionales. Las extensiones se ejecutan en procesos separados, lo que garantiza que no ralentizarán su editor.
  • Implemente con confianza y facilidad Con Microsoft Azure, puede implementar y alojar sus sitios React, Angular, Vue, Node, Python (¡y más!), almacenar y consultar datos relacionales y basados en documentos, y escalar con computación sin servidor, todo con facilidad, todo desde VS Code.
